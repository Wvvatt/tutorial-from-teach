Reference

浏览器视频术语

中英对照速查。课程中尽量使用这里的术语，避免把 Web API、Chromium 内部组件、Linux 平台机制和 DRM 安全域混在一起。

Web API

English中文说明
HTMLMediaElementHTML 媒体元素<video> 和 <audio> 的公共基类，维护状态机和事件。
Media Source Extensions (MSE)媒体源扩展允许 JavaScript 用 SourceBuffer.appendBuffer() 向媒体元素喂数据。
SourceBuffer源缓冲区MSE 中追加 init segment 和 media segment 的对象。
Encrypted Media Extensions (EME)加密媒体扩展标准化网页与 CDM 的密钥会话协议。
MediaKeys媒体密钥对象绑定到 media element 的 CDM 实例抽象。
MediaKeySession密钥会话生成 license challenge、接收 license、维护 key status。

Chromium Media Pipeline

English中文说明
WebMediaPlayerBlink 媒体播放器接口Blink 到 Chromium media 模块的主要抽象边界。
WebMediaPlayerImplWebMediaPlayer 实现选择 progressive/MSE 路径，创建和驱动 media pipeline。
media::Pipeline媒体流水线组织 demuxer、decoder、renderer 的生命周期。
Demuxer解封装器把容器格式拆成带时间戳的 encoded audio/video buffers。
FFmpegDemuxerFFmpeg 解封装器progressive download 路径中的经典 demuxer。
ChunkDemuxer分块解封装器MSE 路径中的 demuxer，接收 JS append 的 bytes。
SourceBufferStream源缓冲流MSE 的 track buffer 层，管理帧、时间窗口、依赖和 eviction。
DecoderBuffer解码输入缓冲压缩后的音视频数据，送入 decoder。
VideoFrame视频帧解码输出的帧抽象，可持有系统内存、GPU mailbox 或 dma-buf。

Decoder

English中文说明
Video Decoder Accelerator (VDA)视频解码加速器旧的 Chromium 硬件解码抽象，输出偏 GPU Picture。
media::VideoDecoder视频解码器接口现代统一解码接口，输出 VideoFrame。
MojoVideoDecoderMojo 视频解码器代理Renderer 进程中的跨进程 decoder 客户端。
CdmContextCDM 上下文decoder 与 CDM 解密/等待密钥逻辑的连接点。
DecryptingVideoDecoder解密视频解码器在 decode 前调用 CDM 解密，常见于 L3 路径。

Compositor

English中文说明
VideoFrameCompositor视频帧合成桥连接 media renderer 与 cc compositor。
cc::VideoLayerImpl视频图层实现在 compositor layer tree 中代表视频帧。
OverlayProcessor叠加处理器分析 DrawQuad 并决定是否提升为硬件 overlay。
OverlayCandidate叠加候选可能被提升到硬件 plane 的 quad 描述。
Hint Frame提示帧提前传递视频帧元数据，辅助 overlay 决策。

Linux Platform

English中文说明
dma-bufDMA 缓冲区Linux 跨设备共享缓冲区机制，以 fd 形式传递。
secure dma-buf安全 DMA 缓冲区普通 CPU/GPU 不可读取的受保护帧缓冲。
KMS / DRM内核模式设置 / 直接渲染管理器Linux 内核显示子系统，管理 connector、CRTC、plane。
Overlay Plane叠加平面Display controller 的独立硬件扫描输出通道。
Scanout扫描输出Display controller 通过 DMA 从 framebuffer 读取像素并输出。
Wayland linux-dmabufWayland dma-buf 协议Wayland 客户端向 compositor 提交 dma-buf 的协议扩展。
V4L2 M2MVideo4Linux2 内存到内存设备嵌入式 SoC 常见的硬件视频解码接口。
GStreamerGStreamer嵌入式浏览器常用媒体框架，可对接 V4L2/dma-buf/EGL。

DRM / Security

English中文说明
Content Decryption Module (CDM)内容解密模块实现 DRM 协议、密钥会话和解密策略的组件。
Widevine L1Widevine 硬件安全等级密钥、解密和帧路径由 TEE/硬件安全保护。
Widevine L3Widevine 软件安全等级CDM 软件解密，安全边界弱于 L1。
Trusted Execution Environment (TEE)可信执行环境与普通 Linux/Android 世界隔离的安全执行环境。
TrustZoneTrustZoneARM 的硬件隔离机制，常用于实现 TEE。
OP-TEE开放 TEE OS常见开源 TEE 实现。
HDCP高带宽数字内容保护HDMI/DP 输出链路保护协议。
Content Protection property内容保护属性Linux DRM connector 上表示 HDCP 状态的属性。
