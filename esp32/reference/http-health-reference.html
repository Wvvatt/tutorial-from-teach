HTTP 状态接口速查

对应 第 5 课。

最小生命周期

#include <WebServer.h>

WebServer server(80);

void setupHttp() {
    server.on("/health", HTTP_GET, handleHealth);
    server.onNotFound([] { server.send(404, "text/plain", "not found\n"); });
    server.begin();
}

void loop() {
    server.handleClient();
}

响应的三个部分

server.send(200, "text/plain", "ok\n");
//          状态码  Content-Type   响应体

200 表示成功；未知路径应返回 404。浏览器和 curl 都是客户端，ESP32 是服务端。

本课状态体

uptime_ms=12345
rssi=-52

handleClient() 和 Wi-Fi 轮询一样：每轮获得一次执行机会，处理一点工作后归还控制权。不要在其他函数里使用长时间阻塞循环。
