V4L2 工具速查表

v4l2-ctl 常用命令

用途命令
列出设备v4l2-ctl --list-devices
全部信息v4l2-ctl -d /dev/video0 --all
设备能力v4l2-ctl -d /dev/video0 --info
枚举 CAPTURE 格式v4l2-ctl -d /dev/video0 --list-formats
枚举 OUTPUT 格式v4l2-ctl -d /dev/video0 --list-formats-out
获取 CAPTURE 格式v4l2-ctl -d /dev/video0 --get-fmt-video
获取 OUTPUT 格式v4l2-ctl -d /dev/video0 --get-fmt-video-out
设置 OUTPUT 格式v4l2-ctl -d /dev/video0 --set-fmt-video-out width=1920,height=1080,pixelformat=H264
列出 controlsv4l2-ctl -d /dev/video0 --list-ctrls
设置 controlv4l2-ctl -d /dev/video0 --set-ctrl=NAME=VALUE
订阅事件v4l2-ctl -d /dev/video0 --subscribe-event=source_change
等待事件v4l2-ctl -d /dev/video0 --wait-event=source_change
抓帧 (MMAP)v4l2-ctl -d /dev/video0 --stream-mmap --stream-count=10 --stream-to=out.raw
送帧 (MMAP)v4l2-ctl -d /dev/video0 --out-mmap --stream-from=in.raw
decoder drainv4l2-ctl -d /dev/video0 --decoder-cmd stop
decoder flushv4l2-ctl -d /dev/video0 --decoder-cmd flush
decoder startv4l2-ctl -d /dev/video0 --decoder-cmd start

v4l2-compliance

命令用途
v4l2-compliance -d /dev/video0基本合规性测试
v4l2-compliance -d /dev/video0 -s含 streaming 的完整测试
v4l2-compliance -d /dev/video0 -m2mmem2mem 设备测试

虚拟驱动

驱动加载命令用途
vividmodprobe vivid通用视频采集/输出模拟
vivid (MPLANE)modprobe vivid multiplanar=2多平面 API 模式
vivid (Request)modprobe vivid supports_requests=2强制使用 Request API
vivid (no inj)modprobe vivid no_error_inj=1关闭错误注入（配合 v4l2-compliance）
vicodecmodprobe vicodecStateless FWHT 虚拟编解码器
vislmodprobe visl多格式 stateless 解码器测试框架

Stateless

  • v4l2slh264dec
  • v4l2slh265dec
  • v4l2slvp8dec
  • v4l2slvp9dec
  • v4l2slmpeg2dec
命名规则：v4l2sl*dec = stateless, v4l2*dec = stateful。

调试工作流

  1. v4l2-ctl --list-devices → 找到目标 /dev/videoX
  2. v4l2-ctl -d /dev/videoX --all → 全面信息
  3. v4l2-compliance -d /dev/videoX -m2m -s → 合规测试
  4. v4l2-ctl -d /dev/videoX --subscribe-event=source_change → 订阅事件
  5. dmesg | grep -i v4l2 → 内核日志
